# The Human Protocol

> The personal side of building big things.

A podcast where builders talk about the messy parts of creating the future. Hosted by [Simona Pop](https://x.com/Sim_Pop).

## Deploy

This is a static site configured for **Vercel**:

1. Push this repo to GitHub
2. Import the repo in [Vercel](https://vercel.com)
3. Deploy — no build step needed

## Structure

```
├── public/
│   ├── index.html      # Main site
│   ├── robots.txt       # Search engine crawling rules
│   └── sitemap.xml      # Sitemap for SEO
├── vercel.json          # Vercel config (headers, redirects, caching)
├── package.json
└── README.md
```

## Domain

Configure `thehumanprotocol.xyz` in Vercel project settings → Domains.
